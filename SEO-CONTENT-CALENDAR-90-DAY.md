# PART 4: SEO TECHNICAL SETUP

### Week 1 Setup (Do This Before Publishing Anything)

| Task | Tool | Time |
|---|---|---|
| Register domain for your agency site | Namecheap / GoDaddy | 15 min |
| Set up hosting | Carrd ($19/mo) or Framer or WordPress | 30 min |
| Install Google Analytics | analytics.google.com | 15 min |
| Install Google Search Console | search.google.com/search-console | 15 min |
| Install Yoast SEO (WordPress) or equivalent | Plugin or built-in | 10 min |
| Submit sitemap to Search Console | search.google.com | 5 min |
| Set up UTM tracking for all links | Google Campaign URL Builder | 10 min |
| **Total** | | **1.5 hours** |

### Ongoing SEO Tasks

| Frequency | Task | Tool |
|---|---|---|
| **Weekly** | Check Search Console for keyword performance, fix any errors | Google Search Console |
| **Weekly** | Check Google Analytics: page views, time on page, bounce rate | Google Analytics |
| **Monthly** | Audit backlinks (who's linking to you?), find link-building opportunities | Ahrefs free / SEMrush free |
| **Monthly** | Update oldest 2 articles with new info + internal links to newer articles | Manual |
| **Quarterly** | Check keyword rankings, adjust strategy | Google Search Console |

### Internal Linking Strategy

When publishing a new article, add links to it from 2-3 older articles (and vice versa). This spreads "link juice" and keeps readers on your site.

| Article # | Internal Links To |
|---|---|
| 1 (Lead Automation Guide) | 5, 7, 8, 14, 22 |
| 2 (WhatsApp CRM) | 9, 18, 21, 23 |
| 3 (Client Onboarding) | 5, 15, 16 |
| 4 (Sokogate) | 1, 19, 20 |
| 5 (Chatbot) | 1, 7, 10, 23 |
| ...and so on |
