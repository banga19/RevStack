## How to Build All 3 in Canva (Step-by-Step)

### Step 1: Set Up Your Brand Kit
1. Go to canva.com → Brand Kit (left sidebar)
2. Add your brand colors:
   - Primary: `#00A859` (Green)
   - Secondary: `#0A1F2E` (Dark navy)
   - Accent: `#C8A96E` (Gold)
   - Neutral: `#F0F4F8` (Light grey)
3. Add your fonts:
   - Headings: Montserrat Bold (one-pager), Playfair Display (proposal), Inter (case study)
   - Body: Open Sans (one-pager), Inter (proposal + case study)
4. Upload your logo

### Step 2: Create Each Template
1. Click "Create a design" → choose the dimensions above
2. Apply your brand colors to backgrounds and text
3. Add placeholder text from the layouts above
4. Add shapes for dividers (thin rectangles for lines, circles for icons)
5. Group elements together for each section
6. Save as a template: File → Save as template → Name it clearly

### Step 3: Create Variations
- **One-pager:** Create both A4 (print) and 1920×1080 (digital) versions
- **Case study:** Create both 1080×1920 (Instagram) and 1920×1080 (LinkedIn/document) versions
- **Proposal deck:** Create the cover as one design, then create 6 more pages as separate designs named "Slide 2", "Slide 3", etc.

### Step 4: Share as Editable Templates
- Click "Share" → "Share link to view"
- Set permission to "Anyone with the link can view"
- Or upgrade to Canva Pro to share as editable templates

## Quick Start: Minimal Viable Version (Build in 20 Minutes)

If you need something NOW, here's the stripped-down version for each:

### One-Pager (5 min)
1. Open Canva → Create blank A4
2. Add your logo top-left
3. Big text center: `"[Company] — Pipeline Audit Summary"`
4. 2 columns: Left = problems (red), Right = solutions (green)
5. Bottom: Big number + your phone number
6. Done. Export PDF.

### Proposal Cover (5 min)
1. Open Canva → Create blank 1920×1080
2. Background: Solid dark navy
3. Logo top-left
4. Center text: `"[Company Name]"` in gold, large font
5. Below: `"Lead Automation Proposal"` in white, smaller
6. Bottom: Date + "CONFIDENTIAL"
7. Done. Export as PNG.

### Case Study (10 min)
1. Open Canva → 1080×1920 blank
2. Top 25%: Green background, "CASE STUDY" badge, headline
3. Next 20%: Grey bg, "Before" + bullet points
4. Next 15%: White bg, "What We Built" + 4 bullet points
5. Next 25%: Navy bg, 3 big numbers
6. Bottom 15%: Quote + "[Your Name]" + QMe link
7. Done. Export PDF.
