# Financial Model — RevStack / Mapato

## Assumptions
- Average client retainer: $2,500/month
- Success fee: 10% of closed deal value
- Variable costs per client: integrations, LLM usage, support time
- Fixed costs: hosting, worker infra, monitoring, legal

## Revenue Milestones
- **Day 0:** $0
- **Day 30:** $2,500 — 1 anchor client
- **Day 60:** $10,500 — 1 anchor + 3 retainers
- **Day 90:** $22,500 — 1 anchor + 4 retainers + fees

## Cost Structure
- Minimum viable production: ~$63-120/month
- Recommended production: ~$200-400/month
- Break-even: ~$100/month MRR
