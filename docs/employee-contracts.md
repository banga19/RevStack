# Employee Contracts — RevStack / Mapato

## Coverage
All hired employees should have signed employment agreements.

## Required Clauses
1. **Role & Reporting Line** — title, department, and reporting structure.
2. **Compensation** — salary, bonuses, review cadence.
3. **Benefits** — leave, insurance, travel allowance (if applicable).
4. **Confidentiality** — aligns with NDA template.
5. **IP Assignment** — incorporated by reference to IP Assignment Agreement.
6. **Termination** — notice period, garden leave, severance.
7. **Jurisdiction** — Kenya law; disputes resolved in Nairobi.
